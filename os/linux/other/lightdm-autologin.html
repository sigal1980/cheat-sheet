LightDM - автологин

Отредактируйте файл конфигурации LightDM и убедитесь, что эти строки не закомментированы и правильно настроены:
/etc/lightdm/lightdm.conf:
[Seat:*]
autologin-user=username
Вы должны быть членом группы autologin, чтобы иметь возможность автоматически входить в систему без ввода пароля:
# groupadd -r autologin
# gpasswd -a username autologin
LightDM выполняет вход в систему, используя сеанс, указанный в ~/.dmrc пользователя, который автоматически входит в систему. Чтобы переопределить этот файл, укажите autologin-session в lightdm.conf:
/etc/lightdm/lightdm.conf:
[Seat:*]
autologin-user=username
autologin-session=session
Список допустимых имен сеансов можно найти, указав /usr/share/xsessions/.desktop для сеансов X и /usr/share/wayland-sessions/.desktop для сеансов Wayland.
Включение интерактивного входа в систему без пароля
LightDM проходит через PAM, поэтому вы должны настроить конфигурацию lightdm в PAM:
/etc/pam.d/lightdm:
#%PAM-1.0
auth sufficient pam_succeed_if.so user ingroup nopasswdlogin
auth include system-login
...
В этом случае вы также должны быть частью групп nopasswdlogin и autologin, чтобы иметь возможность интерактивного входа в систему без ввода пароля:
# groupadd -r nopasswdlogin
# groupadd -r autologin
# gpasswd -a username nopasswdlogin
# gpasswd -a username autologin
